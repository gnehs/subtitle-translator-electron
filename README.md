## Supported subtitle extensions

- `.ass`
- `.srt`
- `.ssa`
- `.vtt` WebVTT

## Features

- Translate subtitle using ChatGPT `gpt-3.5-turbo` & `gpt-4`
- Support multiple languages
- Translation according to the preceding and following sentences
